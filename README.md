# AHU Campus Card Records Scraper and Analyzer

## 项目介绍

这是一个用于爬取安徽大学（AHU）校园卡账单记录的Python项目。该项目通过模拟登录安徽大学智慧安大系统，自动获取一卡通消费记录，并提供数据分析功能，帮助用户了解消费习惯、作息规律等。

## 主要功能

- **数据爬取**：
  - 支持全量爬取和增量爬取两种模式
  - 自动登录并捕获认证信息
  - 智能处理食堂名称映射（北一区→桔园等）
  - 防反爬机制，避免被限制

- **数据分析**：
  - 消费极值分析（最高单笔消费、充值）
  - 作息时间分析（最早/最晚消费时间）
  - 食堂三餐统计（各食堂早餐/午餐/晚餐平均消费和次数）
  - 三餐偏好与时间规律分析
  - 非食堂消费汇总

## 依赖环境

- Python 3.7+
- 外部库：
  - `requests` - 用于HTTP请求
  - `playwright` - 用于自动化浏览器操作
  - `pandas` - 用于数据分析
  - `numpy` - 用于数值计算

## 安装步骤

1. 克隆或下载项目到本地：
   ```bash
   git clone <repository-url>
   cd AHU_card_records
   ```

2. 安装Python依赖：
   ```bash
   pip install requests playwright pandas numpy
   ```

3. 安装Playwright浏览器：
   ```bash
   playwright install
   ```

## 使用方法

1. 运行主程序：
   ```bash
   python main.py
   ```

2. 选择操作：
   - `[1] 增量爬取流水记录` - 仅爬取新数据（推荐）
   - `[2] 全量爬取流水记录` - 重新获取所有数据（耗时较长）
   - `[3] 执行一卡通数据分析` - 基于本地数据进行分析
   - `[0] 退出系统`

3. 首次使用时，程序会自动打开浏览器，请按以下步骤操作：
   - 登录你的智慧安大账号
   - 翻到最下面，左下方点击"余额"（不要点击"一卡通"）
   - 等待程序自动捕获认证信息

4. 爬取完成后，数据保存在 `campus_card_records.csv` 文件中。

## 注意事项

- 爬取过程中会自动添加随机延时，避免被服务器限制
- 认证信息会保存到 `headers_config.json`，下次使用时无需重新登录
- 如果认证失效，程序会自动重新捕获
- 分析功能基于CSV数据，确保数据完整性

## 输出文件

- `campus_card_records.csv` - 账单记录CSV文件
- `headers_config.json` - 认证配置（自动生成）

## 数据格式

CSV文件包含以下字段：
- 交易时间
- 交易类型
- 金额(元)
- 余额(元)
- 商户/地点
- 详情描述
- 流水号
- 订单状态
